# Govs Connect

Govs Connect is a student-run mobile application designed for the community of the Governor's Academy. It allows users to gain essential information, such as schedules and menus, in a more productive and elegant way. Released to the public in 2018, Govs Connect has continually helped teachers and students at Govs, achieving 500+ downloads and 300+ monthly active users, covering 70%+ of the community population.
